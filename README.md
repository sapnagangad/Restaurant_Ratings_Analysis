# Restaurant Ratings Analysis (Power BI Project)

This project analyzes restaurant ratings and customer feedback using **Power BI** to uncover insights about restaurant performance, customer satisfaction, and rating distribution.

The dashboard transforms raw restaurant data into **interactive visualizations** that help understand trends in ratings, cuisines, locations, and restaurant popularity.

---

## Project Objective

The objective of this project is to demonstrate how **Power BI can be used for data analysis and visualization** to generate insights from restaurant rating data.

The project focuses on:

- Analyzing restaurant ratings
- Identifying top-performing restaurants
- Understanding customer preferences
- Exploring rating trends across different cuisines and locations

---

## Dataset Overview

The dataset includes information about restaurants and their ratings.

### Key Columns

| Column | Description |
|------|-------------|
| Restaurant Name | Name of the restaurant |
| Location | City or area of the restaurant |
| Cuisine | Type of food served |
| Rating | Customer rating score |
| Votes | Number of customer votes |
| Price Range | Average price category |

---

## Dashboard Features

The Power BI dashboard provides several interactive insights:

### 1. Overall Rating Overview
Displays the overall distribution of restaurant ratings.

### 2. Top Rated Restaurants
Shows restaurants with the highest ratings and most customer votes.

### 3. Cuisine Analysis
Identifies which cuisines receive the highest ratings.

### 4. Location-Based Analysis
Compares restaurant ratings across different cities or locations.

### 5. Customer Engagement
Analyzes restaurants with the highest number of votes.
